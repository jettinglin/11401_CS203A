# studyNote

本 repository 用來整理我在學習 **資料結構（Data Structures）** 過程中的重點筆記。  
所有內容以 Markdown 撰寫，重點放在 **核心概念、用途與比較**，方便快速複習與回顧。

---

##  檔案結構

目前每一種資料結構皆獨立整理成一份筆記檔案：

- [`array.md`](./array.md)  
  陣列（Array）：連續記憶體、Random Access、基本操作與限制

- [`Linked_List.md`](./Linked_List.md)  
  連結串列（Linked List）：節點結構、插入刪除、與 Array 比較

- [`stack_and_queue.md`](./stack_and_queue.md)  
  堆疊與佇列（Stack / Queue）：LIFO、FIFO、常見應用

- [`hash.md`](./hash.md)  
  雜湊（Hashing / Hash Table）：key-value、collision、load factor

- [`tree.md`](./tree.md)  
  樹狀結構（Tree / BST / Traversal）：階層結構與走訪方式

- [`heap.md`](./heap.md)  
  堆積（Heap）：完全二元樹、Priority Queue、核心操作

- [`Graph.md`](./Graph.md)  
  圖（Graph）：表示法、BFS / DFS、與 Tree 的差異

---

##  使用方式

- **初學**：依序閱讀各筆記，建立資料結構整體觀念  
- **複習**：快速查看各檔案中的重點整理與比較表  
- **實作 / 考試前**：回顧特性與時間複雜度，避免混淆
